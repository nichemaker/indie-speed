## Brief methodology (NON-NEGOTIABLE, from OS)

Before ANY outline, run the 4-step augmented query network pre-brief:
1. **Raw query** (the head term)
2. **Transformation variants** (reformulations Google generates: synonyms, question forms, intent shifts)
3. **Next-step queries** (what the searcher asks after — these become internal links)
4. **Synthetic variants** (zero-volume augmentation targets — one paragraph or table row each)

Each H2/H3 = passage-level standalone answer to ONE variant. Internal links follow next-step query architecture. Niche pages are ARTICLES (package a known answer), not essays.
